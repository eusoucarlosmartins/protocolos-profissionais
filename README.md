# Protocolos Profissionais

Aplicacao web desenvolvida para a Extratos da Terra com foco em consulta e organizacao de protocolos profissionais, produtos e rotinas de atendimento.

## Sobre o projeto

O sistema foi pensado para apoiar o uso interno da equipe e facilitar a consulta de informacoes de protocolos e produtos em uma experiencia simples, responsiva e orientada ao dia a dia comercial.

Entre os recursos da plataforma:

- consulta de protocolos
- busca por produtos
- area administrativa para atualizacao de conteudo
- compartilhamento de informacoes por WhatsApp

## Tecnologias

- React
- Vite
- Supabase
- Vercel

## Execucao local

```bash
npm install
npm run dev
```

Para gerar a build de producao:

```bash
npm run build
```

## Deploy

Publicacao em ambiente Vercel.
